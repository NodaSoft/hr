### Качество кода
Индентация ок, названия переменных понятные, скобки блоков бесят:
то так
```php
public function getRequest($pName)
{
    return $_REQUEST[$pName];
}
```
то так
```php
if ($reseller === null) {
    throw new \Exception('Seller not found!', 400);
}
```
вы уж определитесь
